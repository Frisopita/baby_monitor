# ¿Qué es el proyecto?

## Requerimientos

Dispositivo monitor de bebé:
- Medir ritmo cardiaco
- Medir calidad del sueño
- Medir la poscición del bebé (boca arriba, boca abajo, lado izquierdo, lado derecho)
- Carga inalambrica
- Conexión por medio de BLE
- Conexión a WIFI

App de dispositivo:
- Mostrar los datos
